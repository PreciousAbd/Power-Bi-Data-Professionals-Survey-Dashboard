# Power-Bi-Data-Professionals-Survey-Dashboard
An interactive Power BI dashboard built on a real world survey of 620 data professionals. The report analyzes demographics, salaries, job roles, geographic distribution, work-life balance, salary satisfaction, career entry difficulty, and preferred programming languages.

Project Overview:
This project explores insights from a real life survey of 620 respondents, mostly data analysts, to understand trends across the data profession. The Power BI dashboard visualizes key metrics related to demographics, compensation, job satisfaction, and technical skills.
